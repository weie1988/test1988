俄罗斯方块游戏

这是一个使用Python和Pygame开发的经典俄罗斯方块游戏。由于这是一个桌面应用程序，您需要下载并在本地运行。

下载游戏

游戏特点

  • 七种不同形状的方块（I、O、T、L、J、S、Z）
  • 方块旋转和移动
  • 行消除和分数计算
  • 难度随等级提升而增加
  • 下一个方块预览
  • 游戏暂停功能
  • 背景音乐和音效
  • 高分排行榜

安装与运行

1. 确保您已安装Python 3.6或更高版本

2. 解压下载的文件

3. 打开命令提示符或终端，进入解压后的目录

4. 安装依赖：

pip install -r requirements.txt

5. 运行游戏：

python tetris.py

游戏控制

  • 左右箭头键：左右移动方块
  • 上箭头键：旋转方块
  • 下箭头键：加速下落
  • 空格键：硬下落（立即放置方块）
  • P键：暂停/继续游戏
  • M键：静音/取消静音
